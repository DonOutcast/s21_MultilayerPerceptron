2 Использование калькулятора SmartCalc_v2.0

Использование калькулятора:

  • Для вычисления необходимо набрать нужное математическое выражение и нажать на клавишу "=".
  • Если выражениt содержит "x", то для рассчета необходимо также ввести значение переменной в соответствующее поле ввода перед нажатием клавиши "=".
  • Если выражение будет некорректным, либо результат вычисления будет слишком большим/слишком маленьким/неопределенным, то на дисплее калькулятора отобразится фраза "Incorrect input".
  • Для перехода к режиму построения графика необходимо нажать клавишу "Graph".
  • Для перехода к режиму депозитного калькулятора необходимо выбрать вкладку "DepositCalc".
  • Для перехода к режиму кредитного калькулятора необходимо выбрать вкладку "CreditCalc".
